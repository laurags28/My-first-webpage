Sobre mí

Soy estudiante de Desarrollo de Aplicaciones Web, una disciplina que se enfoca en la creación de aplicaciones interactivas que funcionan a través de navegadores web. Esto incluye el diseño de interfaces de usuario atractivas, la implementación de funcionalidades del lado del cliente y del servidor, y el uso de tecnologías como HTML, CSS, JavaScript y frameworks modernos para desarrollar soluciones web eficientes y escalables.
